Generación de Keypoints (`ml/features/create_keypoints.py`)
============================================================

Este módulo permite generar keypoints a partir de las muestras en video de una palabra
y guardarlos en un archivo HDF5, con el objetivo de preparar los datos para modelos
de machine learning.
