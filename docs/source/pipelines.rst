Pipeline desde Cámara (`ml/features/pipelines.py`)
==================================================

Este módulo conecta todas las etapas del flujo de datos: captura, normalización
y extracción de keypoints. Está pensado para registrar nuevas palabras desde la
cámara web y preparar los datos para entrenamiento de modelos.
