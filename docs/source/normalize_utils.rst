Normalización de Muestras (`ml/utils/normalize_utils.py`)
============================================================

Este módulo ofrece funciones para normalizar secuencias de imágenes (frames) 
capturadas desde muestras de lenguaje de señas. Incluye lectura, interpolación, 
limpieza y guardado eficiente para entrenamiento de modelos.
